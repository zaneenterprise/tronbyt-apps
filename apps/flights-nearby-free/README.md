# Flights Nearby

Flights Nearby is a Tidbyt app that shows the nearest flight around a chosen location.

*** What it does ***

* Lets the user pick a location through a text box
* Shows live ADS-B flight data using adsb.lol free API
* Displays the nearest flight on the left side of the layout
* Shows origin and destination airport when available
* Shows more details when extended details enabled
